## About Me

AI Engineer | Agent Architect | LLM & Recommendation Specialist

2022–2024 NLP Lab, Northeastern University | Focused on **LLM Post-training** and **RAG Systems**  
2024.06–2024.09 Tencent | Built **Multimodal LLMs** for deep POI content understanding  
2024.11–Present Soul | Leading **Recommendation System Scaling** and full-link **Agentic Infrastructure**

My long-term vision:  
**Build fully autonomous systems driven entirely by intelligent agents.**  
Enable agents to self-evolve with LLMs, and use real-world agent operation data to continuously push the boundaries of foundation model capabilities.

I focus on building production-grade LLM agents, autonomous recommendation systems, and end-to-end automation pipelines. I turn complex AI research into clean, repeatable, high-impact engineering systems.
